HTML para Iniciantes

O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Com HTML, você pode criar sua própria página da web. Este exemplo básico mostra como começar:

<!DOCTYPE html>
<html>
<head>
    <title>Meu Primeiro Site</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é um parágrafo.</p>
</body>
</html>

Títulos em HTML

Os títulos são definidos com as tags <h1> a <h6>, onde <h1> é o título mais importante e <h6> é o menos importante.

<h1>Título 1</h1>
<h2>Título 2</h2>
<h3>Título 3</h3>
<h4>Título 4</h4>
<h5>Título 5</h5>
<h6>Título 6</h6>

Links em HTML

Links são criados usando a tag <a>. O atributo href especifica o destino do link.

<a href="https://www.example.com">Visite o Exemplo</a>

Tabelas em HTML

As tabelas são definidas com a tag <table>, e cada linha da tabela é definida com a tag <tr>. Um cabeçalho de tabela é definido com a tag <th>, e uma célula de tabela é definida com a tag <td>.

<table>
    <tr>
        <th>Cabeçalho 1</th>
        <th>Cabeçalho 2</th>
    </tr>
    <tr>
        <td>Dado 1</td>
        <td>Dado 2</td>
    </tr>
</table>

Campos de Formulário em HTML

Os campos de formulário são criados usando a tag <input>. Existem diferentes tipos de campos, como texto, email, senha, etc.

<form>
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome"><br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br>
    <label for="senha">Senha:</label>
    <input type="password" id="senha" name="senha"><br>
    <input type="submit" value="Enviar">
</form>

Botões em HTML

Botões são definidos com a tag <button> ou a tag <input type="button">.

<button>Clique em mim</button>

<input type="button" value="Clique em mim">

Outros Elementos em HTML

Existem muitos outros elementos em HTML, como listas, imagens, vídeos, etc.

Listas

Existem dois tipos de listas: ordenadas e não ordenadas.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

Imagens

Imagens são inseridas com a tag <img>.

<img src="imagem.jpg" alt="Descrição da imagem">

Vídeos

Vídeos são inseridos com a tag <video>.

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    Seu navegador não suporta o elemento de vídeo.
</video>

CSS para Iniciantes

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para estilizar elementos HTML. Com CSS, você pode controlar a aparência de seus sites, incluindo layouts, cores e fontes.

Seletores CSS

Os seletores CSS são usados para selecionar os elementos HTML aos quais você deseja aplicar estilos.

/* Seletor de elemento */
h1 {
    color: blue;
}

/* Seletor de classe */
.nome-classe {
    font-size: 18px;
}

/* Seletor de ID */
#nome-id {
    font-weight: bold;
}

Propriedades CSS Importantes

  • color: Define a cor do texto.
  • font-family: Define a família de fontes para o texto.
  • font-size: Define o tamanho da fonte.
  • background-color: Define a cor de fundo de um elemento.
  • border: Define a borda de um elemento.
  • margin: Define as margens ao redor de um elemento.
  • padding: Define o preenchimento dentro de um elemento.
  • display: Define o tipo de exibição de um elemento.

Estilos em Cascata

O termo "Cascading" em CSS significa que os estilos são aplicados em cascata. Isso significa que um estilo definido em um local pode ser substituído por um estilo definido em outro lugar.

Box Model

O Box Model é uma parte fundamental do CSS que define como os elementos HTML são renderizados e como eles ocupam espaço na página.

Posicionamento

O posicionamento em CSS determina como os elementos são posicionados na página. Existem diferentes valores de propriedade de posicionamento, como static, relative, absolute e fixed.

JavaScript para Desenvolvedores Web

O que é JavaScript?

JavaScript é uma linguagem de programação de alto nível, dinâmica e orientada a objetos amplamente usada para criar e controlar conteúdo dinâmico em páginas da web.

Principais Conceitos do JavaScript

  • Tipos de Dados (Strings, Números, Booleanos, Arrays, Objetos, etc.)
  • Estruturas de Controle (if, else, switch, loops)
  • Funções (Definição, Chamada, Parâmetros, Retorno)
  • Manipulação do DOM (Document Object Model)
  • Eventos (Manipulação de Eventos do Usuário)
  • AJAX (Asynchronous JavaScript and XML)
  • ES6+ (Recursos Modernos da Linguagem)

Manipulação do DOM

O DOM é uma representação em forma de árvore dos elementos HTML de uma página, permitindo que JavaScript interaja com o conteúdo e a estrutura da página.

Eventos JavaScript

Eventos são ações que ocorrem na página web que podem ser detectadas pelo JavaScript, como clicar em um botão, passar o mouse sobre um elemento, pressionar uma tecla, etc.

Frameworks e Bibliotecas JavaScript

Existem muitos outros elementos em HTML, como listas, imagens, vídeos, etc.

Exemplos de JavaScript

Exemplo de Função em JavaScript

// Definição de uma função
function greet(name) {
    return 'Olá, ' + name + '!';
}

// Chamada da função
console.log(greet('Mundo')); // Saída: Olá, Mundo!

Exemplo de Manipulação do DOM

// Seleciona um elemento pelo ID
var element = document.getElementById('myElement');

// Altera o conteúdo do elemento
element.innerHTML = 'Novo conteúdo';

Exemplo de Evento JavaScript

// Adiciona um evento de clique a um botão
document.getElementById('myButton').addEventListener('click', function() {
    alert('Botão clicado!');
});
